# Shopping-mini-game / 쇼핑 미니 게임
> **스택**

1. HTML
2. CSS
3. JavaScript

> **기능**
1. JSON을 이용한 옷(데이터) 정보 받아오기
2. 하단 버튼을 누르면 해당하는 옷 재고 상태를 볼 수 있음
3. STOCK 타이틀 누르면 원래 재고 모습으로 되돌아옴
4. 화면 안에서 위 아래 스크롤 가능
